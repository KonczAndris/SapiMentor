The Informatics program is a 3-year full-time undergraduate course. It is recommended for you if algorithmic thinking is important to you; you are interested in the design and implementation of software systems; you are curious about the latest software technologies (web, mobile, etc.); and you want to learn as many programming languages as possible. Career opportunities include software developer, software engineer, IT project manager, system administrator, computer science teacher, and researcher. Az Informatika szak egy 3 éves nappali alapképzés. Neked ajánljuk, ha fontos számodra az algoritmikus gondolkodás; érdekel a szoftverrendszerek tervezése és kivitelezése; kíváncsi vagy a legújabb szoftvertechnológiákra (web, mobil stb.); és minél több programozási nyelvet szeretnél elsajátítani. Elhelyezkedési lehetőségek: szoftverfejlesztő, programtervező informatikus, szoftvermérnök, IT-projektmenedzser, rendszergazda, informatika tanár, kutató.

The Computer Science program is a 4-year full-time undergraduate course. It is recommended for you if you are fascinated by computers and want to understand how they can perform complex tasks; you are interested not only in programming but also in the internal structure of computers and operating systems; and you enjoy programming not only for computers but also for other computing devices (smartphones, tablets, microcontrollers, signal processors, programmable logic controllers). Career opportunities include software designer or programmer, system integrator or tester, network designer and implementer, designer and implementer of microprocessor circuits and devices, system administrator, educator, and researcher. A Számítástechnika szak egy 4 éves nappali alapképzés. Neked ajánljuk, ha érdekelnek a számítógépek és azt szeretnéd megérteni, hogyan képesek bonyolult feladatokat elvégezni; nem csak programozni tanulnál, hanem a számítógépek belső felépítése és az operációs rendszerek is érdekelnek; szívesen programoznál számítógépek mellett egyéb számítástechnikai eszközöket is (okostelefonok, táblagépek, mikrovezérlők, jelfeldolgozó processzorok, programozható logikai vezérlők). Elhelyezkedési lehetőségek: szoftvertervező vagy programozó, rendszerintegrátor vagy rendszertesztelő, hálózatok tervezője és kivitelezője, mikroprocesszoros áramkörök és eszközök tervezője és kivitelezője, rendszergazda, oktató, kutató.

The Automation Engineering program is a 4-year full-time undergraduate course. You will learn not only programming skills but also delve into the integration of electronics and mechanical systems with information technology. Career opportunities include becoming a control system designer and programmer, automation engineer, robotics engineer, industrial network designer, and researcher in advanced automation technologies. Az Automatizálási Mérnöki szak egy 4 éves nappali alapképzés. Ajánljuk neked, ha érdekelnek az automatizálási technológiák és szeretnéd megérteni, hogyan képesek számítógépek és vezérlőrendszerek hatékonyan automatizálni ipari folyamatokat. Elhelyezkedési lehetőségek: irányítórendszerek tervezője és programozója, automatizálási mérnök, robotika mérnök, ipari hálózatok tervezője és kutató az előrehaladott automatizálási technológiák területén.
